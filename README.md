# tuia-vue-h5

基于vue-cli3.0+的vue2.0+的多页面工程，其中主要使用了TypeScript、Less等

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
